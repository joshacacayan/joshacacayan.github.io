Dean GullBall

Dean GullBall is a 3d based Roll-a-Ball game made in unity. The goal of the game is to find all of the required keys needed in each level to unlock a door.
When each door is open, it leads the player to the next level.
There are four levels in the game that the player has to surpass to finish the game.
I worked on this project with a group of 4, with my part of the project dealing with level design and programming aspects of the project.
